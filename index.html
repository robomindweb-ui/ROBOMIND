🏗️ HTML Temelleri

HTML (HyperText Markup Language), web sayfalarının iskeletini oluşturan işaretleme dilidir. HTML etiketleri kullanılarak içerik yapılandırılır.

Temel HTML Etiketleri:

Başlık Örnekleri (h1-h6)

Bu bir h4 başlığıdır

Bu bir h5 başlığıdır
Bu bir h6 başlığıdır

Bu bir paragraftır. İçinde kalın yazı, italik yazı, altı çizili yazı, vurgulanmış metin ve küçük metin örnekleri bulunmaktadır.

Ayrıca silinmiş metin ve eklenmiş metin etiketleri de vardır.

"Öğrenmenin en güzel yanı, hiç kimsenin sizden alamayacağı tek şey olmasıdır." - B.B. King

HTML kodunu göstermek için <code> etiketini kullanırız.

<html>
    <head>
        <title>Örnek Sayfa</title>
    </head>
    <body>
        <h1>Merhaba Dünya!</h1>
    </body>
</html>

🎨 CSS Temelleri

CSS (Cascading Style Sheets), HTML elementlerinin görünümünü ve düzenini kontrol eden stil dilidir.

CSS'in Üç Türü:

  1. İç CSS (Internal): <style> etiketi içinde yazılır
  2. Satır içi CSS (Inline): style="" özniteliği ile yazılır
  3. Dış CSS (External): .css dosyası olarak yazılır

📐 Layout Teknikleri

Semantik HTML5 Elementleri:

  • <header> - Sayfa veya bölüm başlığı
  • <nav> - Navigasyon menüsü
  • <main> - Ana içerik
  • <section> - Bölüm
  • <article> - Makale
  • <aside> - Yan içerik
  • <footer> - Alt bilgi

CSS Box Model:

Her HTML elementi bir kutudur ve şu bileşenlerden oluşur:

  • Content: İçerik alanı
  • Padding: İç boşluk
  • Border: Kenarlık
  • Margin: Dış boşluk
